前期准备工作

  • 了解 GitHub Pages，GitHub Pages 是面向用户、组织和项目开放的公共静态页面搭建托管服务，我们的个人博客需要挂在到 GitHub Pages上，同时可以管理我门的相应代码。

  • 了解 Hexo，什么是Hexo，Hexo 是一个快速、简洁且高效的 博客框架。Hexo 使用 Markdown（或其他渲染引擎）解析文章，在几秒内，即可利用靓丽的主题生成静态网页。

  • 了解 NexT，Elegant Theme for Hexo 精于心，简于形。一种 Hexo 的前端主题。

  • 个人域名，这个不用多说了

好了，以上了解后，我门就动起手来搭建自己的个人博客吧。
