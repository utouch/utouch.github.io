快到极致的 Android 模拟器Genymotion

发表于 2016-03-20   |     |    人喜欢

快到极致的 Android 模拟器 －官方
一点都不谦虚，你咋不上天呢

安装

由于Genymotion需要依赖运行环境VirtualBox，安装前需要先安装VirtualBox。Genymotion 和 VirtualBox都是一键式安装，而且官网也有详细教程，这里就不重复造轮子了。

VirtualBox 地址：https://www.virtualbox.org/wiki/Downloads/
Genymotion 中文地址：http://www.genymotion.net
Genymotion 英文地址：http://www.genymotion.com

安装好后是这样

你以为这样就好了么，你还需要添加一个虚拟机，当你满心欢喜的点击Add的时候，她却告诉你无法创建

Failed to deploy virtual device
Unadle to create virtual device:
This file is corrupt.

这就好比说，老子裤子都脱了，你却不让老子干。

遇到问题及解决

有问题就有解决办法，你不让我下载，我去添加离线下载总行了吧。首先进入Setting，查看 Virtual devices

这里是我的

/Users/yourMacName/.Genymobile/Genymotion/deployed/

然后定位到.Genymobile下，打开genymotion.log

找到下载的连接地址

http://files2.genymotion.com/dists/5.0.0/ova/genymotion_vbox86p_5.0_151117_231324.ova
注意这个地址对应Add要添加的虚拟机

这样你可以离线下载了，值得注意的是，这个下载地址对应你在前面点击Add选择添加的哪个虚拟机版本，后面会用到。

重要的事要重复说

把下载的文件放到 /Users/yourMacName/.Genymobile/Genymotion/ova 文件夹下，然后点击Add选择前面Add的虚拟机版本
